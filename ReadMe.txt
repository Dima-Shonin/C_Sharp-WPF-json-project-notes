c# + WPF + json

Тема: Программа для создания заметок


Реализовал:

1) Добавление записей
2) Удаление записей
3) Отображение выполнения 
4) Изменение записей
5) Сохранение изменений с помощью сериализации json

Внимание: Чтобы добавить заметку полностью надо 

1) Заполнить пустое поле 
2) После заполнения переключиться на другое чтобы синее выделение пропало с редактироемого поля

Удаление происходит клавишей Delete
